# PlaceholderChat
Format your chat with PlaceholderAPI

Default config:
```yaml
# LuckPerms Placeholders
#  - %luckperms_prefix% - the players prefix
#  - %luckperms_suffix% - the players suffix
#  - %luckperms_primary_group% - the players primary group
#  - %luckperms_groups% - a list of all groups the player is in
#  - %luckperms_meta;KEY% - the value from the meta with the key.

nametag: "%player_displayname%"
chatmessage: "[%player_level%] <%player_nametag%> %message%" # The message in the chat
```
